## 🤹‍♀️Role

수정

```
- UML 분석
- 게임 내 음악 추가
- 게임 설명 버튼, View 추가
- PPT 제작
- 게임 변경 기획
- 게임 UI 디자인
```

승훈

```
- UML 분석
- 버튼 및 화면 클릭 이벤트시 효과음 탐색
- 적정 중력값 선정, 장애물의 적정 속도 선정
- PPT 제작
- 게임 변경 기획 
- 재시작 버튼 이미지 제작
```

주영

```
- UML Diagram 제작 및 분석
- PPT 제작
- 게임 변경 기획 및 발표
```
